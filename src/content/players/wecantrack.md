## Overview

wecantrack is a Netherlands-based affiliate analytics and conversion attribution platform that solves one of the biggest operational headaches in affiliate marketing: data fragmentation. Publishers and affiliate marketers running programs across multiple networks — CJ, Awin, Impact, Rakuten, AvantLink, and hundreds more — have historically been forced to log into each network separately to pull reports, reconcile data, and understand which traffic sources drive which conversions.

wecantrack addresses this by acting as an aggregation layer: connect your affiliate network accounts, install a lightweight JavaScript tag on your website, and the platform consolidates all affiliate data into a single unified dashboard. More importantly, wecantrack closes the attribution loop by pushing affiliate conversion data back into ad platforms (Google Ads, Meta Ads, TikTok Ads, Microsoft Ads), enabling ROAS optimization based on actual affiliate revenue rather than proxy metrics like clicks or page views.

The platform is particularly relevant in 2026 as affiliate marketing increasingly intersects with paid media — publishers running SEO + paid traffic combinations need to attribute affiliate conversions to specific ad campaigns, keywords, and creatives to optimize spend efficiently. wecantrack's core value proposition is connecting those dots across 350+ affiliate networks automatically.

Awin has specifically highlighted wecantrack as a preferred integration for publisher analytics, calling out its ability to crunch data across 350+ affiliate networks as a key capability for publisher diversification strategy.

## How It Works

wecantrack operates through a three-layer architecture:

1. **Data Collection (JavaScript Tag)**: Publishers install a single JS snippet on their site. The tag automatically detects affiliate links, appends click ID parameters, and tracks user sessions through to conversion events.
2. **Network API Connections**: Publishers connect their accounts from 350+ affiliate networks via API keys. wecantrack pulls commission, click, and conversion data from each network on a scheduled basis (typically hourly or daily).
3. **Attribution Matching**: wecantrack matches incoming conversions from affiliate networks to the specific traffic sources, campaigns, ad groups, keywords, and pages that generated the click — providing full-funnel attribution.
4. **Ad Platform Integration**: Conversion data (with revenue values) is pushed back to Google Ads, Meta Pixel, TikTok Ads Manager, Microsoft Ads, and others as custom conversion events, enabling smart bidding algorithms to optimize toward actual affiliate revenue.
5. **Unified Dashboard**: All data surfaces in a single reporting interface with cross-network analytics, traffic source breakdowns, campaign-level ROAS, and content performance metrics.

## Key Products & Services

### Unified Affiliate Dashboard
Aggregates data from 350+ affiliate networks into one interface. Publishers can see total commissions, conversion rates, EPC (earnings per click), and revenue trends across all their programs without logging into multiple network portals.

### Conversion Attribution Engine
The core product. Matches affiliate conversions to specific traffic sources — organic search, paid ads, social, email, direct — at the session and click level. Eliminates the "black box" problem of not knowing which content or campaign drives affiliate revenue.

### Ad Platform Conversion API Integrations
Native integrations with:
- **Google Ads**: Push affiliate conversions as Google Ads conversion events with revenue values
- **Meta Ads (Facebook/Instagram)**: Feed affiliate revenue into Meta CAPI for campaign optimization
- **TikTok Ads**: Affiliate conversion data for TikTok Smart Performance Campaigns
- **Microsoft Ads**: Conversion data for Bing-driven affiliate traffic

### Automatic Link Monetization Script
A secondary product that automatically detects product URLs on publisher sites and converts them to tracked affiliate links, removing manual linking overhead.

### Reporting & Analytics Suite
Custom dashboards, email reports, scheduled exports, advanced API access, webhooks support, BigQuery integration, and Google Data Studio/Looker Studio connectors for enterprise reporting workflows.

### Network Discovery & Diversification Insights
Leverages aggregated data to surface which affiliate networks and programs are performing best for similar publishers, informing program expansion decisions.

## Pricing & Commission Model

wecantrack operates on a SaaS subscription model:

- **Starter**: Entry-level plan for small publishers; limited network connections and data history
- **Professional**: Core plan for active affiliate publishers; full network integrations, ad platform connections, custom reports
- **Enterprise**: Custom pricing for large publishers and agencies; dedicated support, advanced API access, custom integrations

Exact pricing tiers are not publicly listed; prospective customers are directed to request a demo for custom pricing based on traffic volume and network count. Third-party sources suggest plans range from ~$50/month to $500+/month depending on scale.

No commission-sharing model — wecantrack is a pure SaaS fee business.

## Strengths

- **Breadth of Network Coverage**: 350+ affiliate network integrations is among the highest in the industry. Publishers using niche or regional networks (not just the top 5) are covered.
- **Ad Platform Attribution Closure**: The ability to push actual affiliate revenue back into Google/Meta/TikTok smart bidding is a genuine competitive advantage. Most affiliate analytics tools stop at the dashboard; wecantrack closes the loop.
- **JavaScript-Based Auto-Tracking**: One-tag implementation that automatically detects and tracks affiliate links dramatically reduces publisher setup time compared to manual link-by-link tagging.
- **Awin Partnership**: Being called out by Awin — one of the world's largest affiliate networks — as a preferred analytics integration adds significant credibility and distribution.
- **Publisher-Centric**: Designed specifically for publishers (content sites, comparison sites, review sites), not brands or networks. Feature set maps precisely to publisher pain points.
- **Privacy-Compliant Architecture**: European company with GDPR-compliant data handling by default — relevant for publishers with EU audiences.

## Weaknesses

- **Publisher Focus, Not Brand Focus**: wecantrack solves publisher analytics problems; it doesn't directly help brands or vendors manage their affiliate programs. Vendor relevance is indirect.
- **Pricing Opacity**: Not listing prices publicly creates friction in the evaluation process, particularly for smaller publishers.
- **Dependent on Network API Access**: Data freshness and completeness is limited by the quality of each affiliate network's API. Networks with poor APIs create gaps.
- **Category Crowded with Larger Players**: Competes against Affluent, Trackdesk, and affiliate analytics features within Impact and Awin themselves. Differentiation primarily comes from network breadth.
- **Limited Brand-Side Features**: No program management, publisher recruitment, or outreach tools — purely an analytics/attribution layer.
